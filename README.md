# hse-software-design-cli

## Описание репозитория

Данный репозиторий содержит интерпретатор командной строки. Язык реализации - Java.

## Возможности

1. Простые операции

* cat [FILE]

* echo

* wc [FILE]

* pwd

* exit

2. Поддержка одинарный и двойных кавычек

3. Окружение

4. Подпрограммы

5. Оператор "|"
